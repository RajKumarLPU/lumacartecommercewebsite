# LumaCart

A responsive e-commerce starter built with React on the frontend and Node.js/Express on the backend.

## Project structure

- `client/` - React + Vite storefront UI
- `server/` - Express API with sample product data

## Run locally

1. Install dependencies:
   - `npm install`
   - `npm install --prefix client`
   - `npm install --prefix server`
2. Start both apps:
   - `npm run dev`

Frontend runs on `http://localhost:5173` and the API runs on `http://localhost:5000`.
